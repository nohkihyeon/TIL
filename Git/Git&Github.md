# Git & Github

## Git
- 로컬에서 관리되는 버전 관리 시스템(VCS : Version Control System)
- 소스코드 수정에 따른 버전을 관리해주느 시스템

## Github
- 클라우드 방식으로 관리되는 버전 관 시스템(VCS)
- 자체 구축이 아니 빌려쓰는 클라우드 개념
