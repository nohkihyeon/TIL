# [Windows] cmd에서 clear ls 사용법
## 오류내용
![image](https://user-images.githubusercontent.com/65120581/131278118-33840c1f-6c22-4f2f-b742-1a43d9cfcad3.png)
- ls와 clear가 실행할 수 없다.

## 해결방법
doskey 사용 <br>
ex1) doskey ls = dir <br>
ex2) doskey clear = cls
